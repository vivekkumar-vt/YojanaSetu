# 🇮🇳 Yojana Setu (योजना सेतु)

## Overview

Yojana Setu is a full-stack web application that helps citizens discover government welfare schemes based on their profile information such as age, income, gender, state, and marital status. The platform provides intelligent scheme recommendations, eligibility matching, and an interactive chatbot assistant.

## Features

* Smart Scheme Recommendation System
* Eligibility Matching and Percentage Calculation
* Government Scheme Search and Filtering
* Interactive Chatbot Assistant
* Responsive User Interface
* Document Checklist and Bookmarking
* Support for Central and State Government Schemes
